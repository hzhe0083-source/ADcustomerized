## 3. 功能模块与页面清单

- 在线下单系统：产品展示、价格计算、购物车、订单提交
- ERP管理系统：订单管理、客户管理、物料库存、BOM管理
- MES生产系统：工单管理、生产排程、工序流转、实时看板
- 设备管理：设备监控、数据采集、故障报警、维护管理
- 专业模块：色彩管理、拼版集成、后道工艺处理

### 页面与功能
| 页面名称 | 模块名称 | 功能/要点 |
|-----------|-------------|-------------|
| 首页 | 产品分类导航 | 展示UV卷材、喷绘布等产品分类，支持筛选 |
| 产品详情页 | 产品信息展示 | 图片、规格参数、价格、库存；实时价格计算 |
| 购物车页面 | 购物车管理 | 添加/删除、数量修改、总价计算 |
| 订单确认页 | 订单信息填写 | 收货地址、配送方式、备注 |
| 订单管理页 | 订单列表/详情 | 状态筛选、搜索、分页、金额明细、状态更新 |
| 客户管理页 | 客户列表/详情 | 搜索、筛选、导出、历史订单 |
| 库存管理页 | 物料与BOM | 入库/出库/盘点，产品-物料消耗关系 |
| 生产管理页 | 工单与工序流转 | 排程、分配、扫码/点击完工自动流转 |
| 设备监控页 | 设备状态与采集 | 运行状态、墨水余量、错误报警、日志报表 |
| 实时看板页 | 生产看板 | 进度、设备状态、生产效率（大屏） |
| 色彩管理页 | ICC配置 | 设备/材料ICC文件管理 |
| 拼版集成页 | 文件处理 | 对接拼版软件，自动发送与处理 |
| 后道工艺页 | 工艺配置 | 覆膜、包边、留白等参数 |
